**getStudentUDSetup**
----
	Returns structured UD setup data in JSON format.
  
* **Version History:**

  TASS v59.11 - New endpoint for Consent2Go integration.

* **Version:**

  3

* **Permission:**

  Student Records > Student Records Setup > View

* **Method:**

  `GET | POST`
  
*  **Params:**

   **Required:**

   `includelookups [boolean]` - Include Look Up Details

   **Optional:**

   none
 
   **Conditional:**
 
   none
