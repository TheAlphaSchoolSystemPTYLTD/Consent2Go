**getStudentUDArea**
----
	Returns structured UD area data in JSON format.
  
* **Version History:**

  TASS v59.10 - New endpoint for Consent2Go integration.

* **Version:**

  3

* **Permission:**

  Student Records > Students > UD Areas tab > View

* **Method:**

  `GET | POST`
  
*  **Params:**

   **Required:**

   `code [string]` - Student Code

   `areacode [string]` - Area Code
   
   **Optional:**

   none
 
   **Conditional:**
 
   none
